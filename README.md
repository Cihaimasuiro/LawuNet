1.  Clone or download this repository to your computer.
2.  Open Apache NetBeans.
3.  Go to `File > Open Project...`
4.  Navigate to and select the cloned `LawuNet` folder.
5.  Click the "Run Project" button (the green play icon) to build and run the application.
